# Flipkart CSAT Prediction using Machine Learning

## Project Overview

Customer Satisfaction Score (CSAT) is an important metric used by e-commerce companies to measure how satisfied customers are with their services.
This project aims to predict the **Customer Satisfaction Score (CSAT)** using Machine Learning techniques based on various customer and order related features.

The system analyzes customer data and builds a predictive model that can determine whether a customer is satisfied or not.


## Objectives

* Analyze customer satisfaction data.
* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA).
* Train machine learning models.
* Evaluate model performance and accuracy.



## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn



## Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest

These models were trained using the dataset to predict customer satisfaction.



## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation
7. CSAT Prediction

## Conclusion

This project demonstrates how Machine Learning can be used to predict customer satisfaction in e-commerce platforms. The model helps businesses analyze customer feedback and improve their services.
